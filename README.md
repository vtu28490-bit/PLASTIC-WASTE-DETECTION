# Plastic Waste Detection using YOLOv8 ♻️

This project detects plastic waste in images and real-time video using YOLOv8 object detection.

## Features
- Plastic vs Non-plastic detection
- Real-time webcam detection
- Bounding box visualization
- High accuracy object detection

## Technologies Used
- Python
- OpenCV
- YOLOv8 (Ultralytics)
- NumPy

## Installation

pip install -r requirements.txt

## Training

python train.py

## Detection

python detect.py

## Dataset

Custom dataset with annotated plastic waste images.

## Applications
- Smart waste management
- Recycling plants
- Ocean plastic monitoring
- AI smart bins
